# GPT-4-SUPPORT-GOOGLE-SEARCH

## Installation
```bash
git clone https://github.com/DavidGlusch/gpt-4-search.git
```

```bash
python -m venv venv
source venv/bin/activate
pip install -r requirements.txt
```

### Set up .env file
```bash
cp .env.sample .env
```

### Run the app
```bash
flask run
```

LINK TO THE APP: http://127.0.0.1:5000/
